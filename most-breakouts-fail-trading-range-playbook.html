引言

区间最会折腾人。

每一次向上突破都像趋势启动，每一次向下突破都像要崩。你追一下，被打回来；反手追一下，又被拉回去。做一天以后，亏钱的根源经常是一直用趋势日的方式交易区间。

我现在处理区间，先默认一件事：多数突破都会失败。

这个默认值很有用。它会逼你先看突破质量，再决定追、等回踩，还是等失败突破。

区间要用区间脑子

趋势里，顺着方向等回调，逻辑很直接。

区间里，价格在一个范围内反复交换主导权。边缘位置会吸引追突破的人，也会吸引反向做的人。突破力量只要不够强，就很容易被拉回去。

所以区间里最值钱的问题通常是：

  • 现在在边缘还是中间？
  • 这个突破够不够强？
  • 突破后有没有回踩确认？
  • 失败突破有没有成立？
  • 目标该放到区间中部，还是另一边？

只盯“突破了没有”，很容易追在最差的位置。

几个词先说清楚

BO

BO 是 breakout，价格突破区间边缘、前高前低、趋势线或盘整区域。

突破只是尝试。它需要后续跟进，才有资格升级成趋势。

BP

BP 是 breakout pullback。

价格先突破，再回踩原来的突破位，回踩没有失败，随后顺着突破方向继续走。

我更建议新手等 BP，因为它会自动过滤掉很多弱突破。

fBO

fBO 是 failed breakout。

价格先突破边缘，随后回到区间内部，反向开始接管。区间里好做的机会，很多都来自 fBO。

DP

DP 可以理解成双底回调做多，或者双顶回调做空。

它常出现在区间边缘、失败突破之后、某一端多次尝试都没有走出去的时候。

为什么突破经常失败

区间代表多空暂时平衡。

价格走到上沿，会有人追多，也会有人做空；走到下沿，会有人追空，也会有人做多。双方都在等边缘位置。

突破弱时，价格一回到区间内部，追突破的人会止损或平仓，反向力量也会跟上。这一来一回，就容易把价格推向区间另一边。

所以我在区间里看到突破，第一反应是判断强弱。

强突破长什么样

一个突破值得尊重，通常要有这些条件：

  • 突破棒实体明显，有一半以上站在边界外；
  • 收盘靠近突破方向；
  • 后面有 follow-through；
  • 最好是第二次突破尝试；
  • 对手方反扑很弱；
  • 回踩时没有立刻回到区间内部。

这种突破可以考虑等 BP。

我仍然不喜欢追第一下。突破本身再强，第一下追进去也经常面临止损大、盈亏比差的问题。

弱突破更像 fBO 候选

弱突破通常有这些特征：

  • 只探出去一点；
  • 实体很小；
  • 影线很长；
  • 收盘又回到边界附近；
  • 后面马上出现强反向 bar；
  • 第二根没有 follow-through。

这种突破我会当 fBO 候选看。

尤其在区间上沿或下沿，弱突破失败以后，经常会去测试区间中部，甚至另一端。

BP 适合新手

如果突破真的强，市场通常会给你回踩确认。

等 BP 有几个好处：

  • 止损更自然；
  • 能过滤弱突破；
  • 入场位置比追第一下更舒服；
  • 后续目标更容易规划。

比如向上突破以后，价格回踩原区间上沿，回踩没有跌回区间内部，随后出现多头信号棒。这种位置比突破棒当下追进去更容易执行。

fBO 是区间里的主菜

区间交易里，我最重视 fBO。

一个可做的 fBO，通常需要这几步：

  1. 价格突破区间边缘；
  2. 突破没有站稳；
  3. 出现清楚的反向信号；
  4. 价格重新回到区间内部；
  5. 反向开始有 follow-through。

重点是“重新回到区间内部”。只是在边缘晃一下，还不够。

fBO 的目标要务实。很多时候先看区间中部，再看另一边。区间 setup 用趋势日目标去要求，持仓会很难受。

DP：边缘位置的确认

DP 的价值在于它通常发生在更好的位置。

做多时，区间底部先向下试一次，拉回后又试第二次，第二次依然没走出去，随后出现向上的回调买点。这就是双底回调的味道。

做空时，区间顶部反过来看。

我喜欢 DP 的原因：

  • 位置接近区间边缘；
  • 止损可以放在结构外；
  • 至少有机会打到区间中部；
  • 如果区间够大，另一边也有空间。

区间中间少动

区间中间最难赚钱。

原因很现实：

  • 离上沿远；
  • 离下沿也远；
  • 止损不好放；
  • 目标空间小；
  • 多空都容易被下一波来回震出去。

我会把区间分成三块：上沿、下沿、中间。上沿找空的机会，下沿找多的机会，中间主要观察。

中间位置手痒，通常是当天亏损的开始。

止损和目标

BP

做 BP，止损通常放在回踩结构另一侧。

向上突破后做多，价格不该重新深度跌回区间内部。向下突破后做空同理。

fBO / DP

做 fBO / DP，止损通常放在失败突破的极值之外。

如果失败突破成立，价格不该重新突破那个极值并继续扩张。

目标

区间目标要现实：

  • 区间中部；
  • 区间另一端；
  • 测量目标；
  • 前一段震荡的对称位置。

我会少留很大的 runner。区间行情里，利润来得快，回吐也快。

下单前我会问

  • 当前是趋势，还是区间？
  • 我的位置在边缘，还是中间？
  • 这个突破强不强？
  • 突破以后有没有 BP？
  • 失败突破是否已经回到区间内部？
  • DP 的位置是否足够靠近边缘？
  • 止损放在结构外以后，风险合不合理？
  • 目标放到中部还是另一端？

我会避开的坑

看到突破就追

先验突破质量。强突破等 BP，弱突破当 fBO 候选。

区间中间不断开仓

中间位置盈亏比最差。把注意力拉回边缘，交易会少很多，也干净很多。

把每次突破都当趋势启动

突破要有强实体、强收盘、后续跟进、弱反扑。缺这些条件，就按试探处理。

fBO 刚有迹象就抢

等价格回到区间内部，再看反向信号和 follow-through。

区间单幻想全天大趋势

区间单目标要务实。先看中部，再看另一端。拿太远，经常把好单拿坏。

最后

区间的核心纪律很简单：边缘才有价值，中间少动；强突破等回踩，弱突破防失败。

只要先接受多数突破会失败，很多交易动作会自然变慢。慢下来以后，你才会开始区分 BO、BP、fBO 和 DP。

我的建议是：区间里少追第一下。把钱留给边缘位置、失败突破和回踩确认。
