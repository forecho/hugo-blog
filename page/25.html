一般模式：移动光标的方法（#表示常用）

  • h -> 光标向左移动一个字符#
  • j -> 光标向下移动一个字符#
  • k -> 光标向上移动一个字符#
  • l -> 光标向右移动一个字符#
  • n -> n表示数字，如果想进行多次移动的话还可以加上数字，例如10j。
  • [Ctrl]+[f] -> 屏幕向下移动一页，相当于[Page Down]按键#
  • [Ctrl]+[b] -> 屏幕向上移动一页，相当于[Page Up]按键#
